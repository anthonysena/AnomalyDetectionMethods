We developed a data-driven framework to learn biologically plausible value ranges for measurement concepts within the OMOP Common Data Model (CDM) by integrating multiple univariate statistical methods with machine learning–based anomaly detection. Using a single measurement concept, we analyzed aggregated measurement values and their frequencies, assuming harmonized units and no initial stratification by demographic or care setting variables. Five univariate outlier detection methods—Tukey fences, quantile thresholds, Z-scores, modified Z-scores, and the generalized extreme Studentized deviate (ESD) test—were applied independently to characterize the distribution of measurement values. To reduce dependence on any single statistical assumption, we implemented a consensus-based voting approach in which each value received an outlier score corresponding to the number of methods that flagged it as anomalous. Values not consistently identified as outliers were retained to estimate a preliminary plausible range, defined using extreme quantiles of the retained distribution to ensure robustness to noise and distributional irregularities. As a parametric baseline, Gaussian anomaly detection was applied to model the overall distribution and identify values with low probability under a normality assumption. To capture more complex and non-linear patterns, we then applied Isolation Forest using engineered features that describe each value’s position relative to the learned distribution, including scaled value, percentile, distance to the plausible range boundaries, consensus score, and log-transformed frequency. Final anomaly characterization combined signals from both statistical consensus and machine learning, enabling classification of values into plausible, potentially implausible, and structurally unusual categories.

We anticipate that the consensus-based approach will yield a plausible value range that is more stable and robust than thresholds derived from any single method and that extends beyond traditional clinical reference ranges while remaining biologically consistent. Compared to Gaussian-based thresholds, which are sensitive to distributional assumptions, the consensus-derived range is expected to better accommodate skewed and heavy-tailed measurement distributions commonly observed in real-world data. Isolation Forest is expected to identify additional anomalies not captured by univariate methods, particularly low-frequency or structurally isolated values near distribution boundaries, thereby providing complementary information beyond threshold-based approaches. Together, these methods are expected to improve differentiation between implausible measurement values and rare but valid observations, supporting a more nuanced and scalable approach to data quality assessment in observational health data and offering potential integration with tools such as the OHDSI DataQualityDashboard.
